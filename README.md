# laragon

Add [scoop](https://scoop.sh) support for [laragon](https://laragon.org)

## note

Execute using `PowerShell >= 5`

## install

```ps1
inmo laragon -Scope CurrentUser
```

## example

```ps1
# Install the latest version of mariadb
inla mariadb
# Install v1.16.1 version of nginx
inla nginx@1.16.1
# Uninstall apache
unla apache
```
